Detailed Description

Store contextual data associated to one point-to-point point transport.

Currently handles:

  • A map of local and remote capabilities. Overload advertiseCapabilities() to perform the actual sending of local capabilities to the remote endpoint.
  • A MetaObject cache so that any given MetaObject is sent in full only once for each transport stream.
