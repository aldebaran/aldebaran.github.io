00047   /* Represent the serialisation signature of a Type.
00048   * pseudo-grammar:
00049   * root: element
00050   * element: signature annotation.opt
00051   * sinature:
00052   *   | primitive  // in bcCwWiIlLfdsmro, see Type
00053   *   | [element]
00054   *   | {elementelement}

00056   * elementsequence: a list of 1 or more elements
00057   * annotation.opt: empty or <annotation>
00058   * annotation: may contain arbitrary content except \0
