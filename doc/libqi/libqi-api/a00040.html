Detailed Description

A class to perform benchmarks. 

  * This class is used to perform benchmarks.
  *
  * Use:
  * .. highlight:: c++
  *    :linenothreshold: 1
  *    DataPerfSuite DPS("My_project, "My_executable", DataPerfSuite::OutputType_Normal, "data.xml");
  *    { // ...
  *    }
  *    DataPerf DP;
  *    { // ...
  *    }
  *    while (true) {
  *      DP.start();
  *      { // ...
  *      } // What I want to measure.
  *      DP.stop();
  *      { // ...
  *      }
  *      DPS << DP;
  *    }
