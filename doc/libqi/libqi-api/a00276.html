virtual bool qi::StructTypeInterface::canDropFields ( void *  storage,
const std::vector< std::string > &  fieldNames 
) [inline, virtual]

Return whether struct accepts field-name-based conversion that drops fieldNames.

Versioning support.

Conversion between non-equivalent structs will be attempted if all fields are named: fields with matching names will be automatically mapped to each other. canDropFields() will be called on the source to ask if fields given as argument can be dropped (because they do not exist on the target).

fillMissingFields(fields, missing) will be called on the target, with a map of fields that were converted, and the list of missing field names. The function must fill fields with a value for each of the missing fields, or return false (no storage is provided, because the struct cant be instanciated without a value for all fields being available.
