qi.Session API¶

Introduction¶

A session connect to a standalone qi.Session. Once connected the session can:

  • advertise new services using qi.Session.registerService()
  • get services using qi.Session.service()

Reference¶

class qi.Session¶
close() → None¶

Close the Session.

connect(url) → None¶
Parameters:url – string. Address of the ServiceDirectory of the form ‘tcp://IP:PORT‘.
Raise :a RuntimeError if the connection failed.

Connect the session to a ServiceDirectory.

endpoints() → list¶
Returns:the current list of endpoints of the session.
isConnected() → bool¶
Returns:true if the session is connected
listen(url) → None¶
Parameters:url – string. Address of the form ‘tcp://IP:PORT‘.
Raise :a RuntimeError if the listening failed.

Listen for connections on that specific Url.

listenStandalone(url) → None¶
Parameters:url – string. Address of the form ‘tcp://IP:PORT‘.
Raise :a RuntimeError if the listening failed.

Create a session with a standalone ServiceDirectory that will listen for connections on the specified Url.

registerService(name, object) → int¶
Parameters:
  • name – string. A human readable name associated to the service.
  • object – a python Object.
Returns:

the id associated to the service in the ServiceDirectory. This id can be used to unregister the service.
Raise :

a RuntimeError if the service is already registered.

Register an Object as a service on the ServiceDirectory. The name should be unique on the ServiceDirectory.

service(name) → Object¶
Parameters:name – string. The human readable name of the service we want to get.
Returns:an Object representing a Service. The service could have been registered to the ServiceDirectory by this session or by another.
Raise :a RuntimeError if the service is not found.
services() → list¶
Returns:the list of all services registered on the ServiceDirectory.
unregisterService(id) → None¶
Parameters:id – int. The id associated to the service when it was registered.
Raise :a RuntimeError if the service is not found.

Unregister a service. The service should be owned by the current session. Use the Id returned by registerService.

url() → str¶
Returns:the url the session is connected to
Raise :a RuntimeError if the session is not connected
waitForService(name) → void¶
Parameters:name – string. A human readable name associated to the service.

Wait until a service is available.

Examples¶

Getting a service:

import qi

s = qi.Session("tcp://127.0.0.1:9559")
foo = s.service("Foo")

A more explicit example:

import qi

s = qi.Session()
s.connect("tcp://127.0.0.1:9559")

foo = s.service("foo")

Registering a service:

import qi

#sample service doing nothing
class Foo:
  pass

s = qi.Session()
s.connect("tcp://127.0.0.1:9559")

serviceId = s.registerService("Foo", Foo())
